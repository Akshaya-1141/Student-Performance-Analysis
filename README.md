# Student-Performance-Analysis
Student Performance Predictor
Internship Project: Linear Regression Modeling

# This repository contains a machine learning system designed to predict a student's final exam score based on their daily study habits and academic history.

# Project Overview
The goal of this project is to use Linear Regression to analyze how different factors like study hours and attendance impact final marks. It identifies key trends and provides a tool for instant performance estimation.

# Features
- Data Analysis: Visualizes the correlation between study time and scores.
- Machine Learning: A Linear Regression model trained on 1,000 rows of student data.
- Interactive Tool: A simple input system to predict scores for any student.
- Realistic Logic: Scores are automatically capped at 100% to ensure logical results.

# Technologies
- Language: Python
- Libraries: Pandas, Scikit-Learn, Seaborn, Matplotlib
